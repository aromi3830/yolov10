#####구구단 만들기
```dash
for i in range(1, 10):  # 1부터 9까지 반복
    for j in range(1, 10):  # 1부터 9까지 반복
        print(f"{i} x {j} = {i*j}", end="\t")  # i와 j의 곱을 출력
    print()  # 각 단마다 줄 바꿈
```dash
